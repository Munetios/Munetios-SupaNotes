# Munetios SupaNotes

**Munetios SupaNotes** is a powerful, end-to-end encrypted note-taking Progressive Web App (PWA) built for productivity and privacy. Organize your thoughts effortlessly with a clean, responsive interface.

---

## Features

- **End-to-End Encryption** — Your notes are encrypted before they ever leave your device.
- **Note Management** — Create, edit, and delete notes with a rich formatting toolbar.
- **Favorites** — Star notes to keep your most important content one click away.
- **Trash** — Safely move notes to trash before permanent deletion.
- **Tags** — Organize notes with a flexible tagging system.
- **Search** — Instantly search across all your notes, with full mobile support.
- **Light & Dark Mode** — Seamlessly switch between themes.
- **Responsive Design** — Fully optimized for desktop, tablet, and mobile devices.
- **PWA Support** — Install SupaNotes as a standalone app on any device.

---

## Tech Stack

- **Frontend** — Vanilla HTML, CSS, and JavaScript
- **Styling** — [Beautiful CSS](https://api.munetios.com/beautiful-css/beautiful.css) by Munetios · Google Sans Flex · Material Symbols Rounded
- **Security** — Content Security Policy (CSP) enforced; end-to-end encryption
- **API** — Powered by [Munetios API](https://api.munetios.com)

---

## Getting Started

SupaNotes is a single-file web application. No build step is required.

1. Clone or download this repository.
2. Open `index.html` in any modern browser.
